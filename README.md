## What is LinClaw?

**LinClaw** (邻龙虾) is **the first mobile app built for OpenClaw** — turning your phone into a real-time control center for AI digital workers. No terminal, no code, no jargon.

While OpenClaw has Telegram, Discord, and WhatsApp channels for chat, **no one has built a dedicated mobile app** — until now. LinClaw is a native OpenClaw Channel Plugin that goes far beyond messaging: smart status cards, one-tap approvals, cost tracking, and collaborative observation — purpose-built for founders, managers, and business owners who want to manage autonomous AI agents on the go.

### 🎯 Core Value

- **For Bosses**: Manage your AI digital workers like checking food delivery orders — simple status cards, one-tap approvals, cost dashboards.
- **For Developers**: The first and most capable mobile channel for OpenClaw. Full event streams, diff reviews, cross-device sessions.
- **For Teams**: Invite collaborators to observe AI workers in real-time. Transparent, auditable, collaborative.

## ✨ Features

| Feature | Description |
|---|---|
| 📱 **Mobile-First Control** | Flutter App (iOS/Android) |
| 🔌 **Native OpenClaw Plugin** | Install with one command: `openclaw plugins install linclaw` |
| 🧠 **Smart Translation** | Server-side AI adapter translates technical output into plain language |
| 🃏 **3-State Card System** | In Progress · Needs Action · Completed — that's all a boss needs |
| 💰 **Cost Dashboard** | Real-time token spend tracking, daily/weekly trends, per-worker breakdown |
| 👁️ **Collaborative Observation** | Invite team members or clients to watch AI workers in real-time (read-only) |
| 🔔 **Push Notifications** | APNs + WeChat subscription messages for critical decisions |
| 🛡️ **Zero Config NAT Traversal** | Plugin initiates outbound WebSocket — works behind any firewall |
